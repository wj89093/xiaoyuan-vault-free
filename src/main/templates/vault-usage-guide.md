# 晓园 Vault 使用说明

> 欢迎！这是你的 AI 原生知识库。

## 🚀 快速开始（5 分钟上手）

### 第一步：拖入文件
把文件/文件夹拖进窗口，文件自动进入 `_raw/` 来源目录

### 第二步：AI 自动整理
点击右下角 🤖 气泡，和 AI 对话：
- 「帮我分析这些内容」
- 「建议一个目录结构」
- 「这篇的核心观点是什么」

### 第三步：浏览知识
AI 整理后的内容在左侧「知识」Tab，可以：
- 双击打开页面
- 使用搜索（⌘⇧F）找内容
- 点开图谱（🕸️）看页面关系

---

## 💡 核心理念

**你是策展人，AI 是编辑。**

- 你：负责来源策展、提出问题、确认方向
- AI：负责整理、交叉引用、归档、更新页面

## 📁 文件夹结构

```
_raw/       ← 原始资料（AI 只读不修改）
_wiki/      ← 知识页面（AI 全权维护）
_schema/    ← 领域规范（你 + AI 协商制定）
_lint/      ← 健康报告（死链/孤立页面检测）
_briefing/  ← AI 自动生成的简报
```

## 🧭 左侧导航

| 图标 | 功能 |
|------|------|
| 📋 索引 | 打开 index.md 导航所有知识 |
| 🔍 搜索 | 全局搜索（⌘⇧F） |
| 🕸️ 图谱 | 可视化页面关联 |
| 📤 产出 | AI 生成内容的输出面板 |
| ✅ Lint | 检查死链/矛盾/孤立页面 |
| 📝 日志 | 查看操作记录 |
| 🗂️ Schema | 管理领域字段规范 |
| 📊 简报 | AI 自动生成定期总结 |
| ⚙️ 设置 | 系统配置和账户 |
| 🗑️ 回收站 | 恢复已删除的文件 |

## ⌨️ 快捷键

| 操作 | 快捷键 |
|------|--------|
| 搜索 | ⌘⇧F |
| AI 对话 | ⌘I |
| 导入文件 | ⌘⇧I |
| 保存 | ⌘S |
| 快速切换 | ⌘⇧O |

> **小贴士**：和 AI 对话时，尽量描述你想要的输出，而不是问"怎么做"。比如"帮我把这篇论文整理成摘要"比"怎么做摘要"效果更好。
